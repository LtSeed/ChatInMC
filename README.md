# ChatInMC

ChatInMC是一个基于Bukkit的Minecraft聊天机器人插件，通过对话流API实现与玩家的互动。

**依赖**

- Bukkit API
- Reflections
- Lombok
- Google Cloud Java SDK

**功能**

- 支持在游戏中创建聊天机器人并进行对话。
- 支持在游戏中管理已创建的聊天机器人。
- 支持自定义聊天机器人的语言模型。

**安装**

1. 下载插件到Bukkit插件目录。
2. 下载和安装Google Cloud SDK和DialogFlow API。
3. 在config.yml中进行插件配置。

**使用**

在游戏中使用以下命令：

- **/cim create** 创建一个新的聊天机器人。
- **/cim manage** 管理已创建的聊天机器人。
